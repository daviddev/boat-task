### After cloning repository run these commands.
````
composer install
````
````
copy .env.example .env
````
````
set STRIPE_KEY and STRIPE_SECRET in .env file
````
````
set DB_DATABASE, DB_USERNAME and DB_PASSWORD in .env file
````
````
php artisan migrate
````
````
php artisan db:seed
````
````
npm install
````
````
npm run dev
````
